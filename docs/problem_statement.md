Background (Real-World Context)
In hospitals and clinics, doctors write clinical notes describing:
    Patient symptoms
    Diagnoses
    Procedures performed

These notes must be converted into standard medical codes such as:
    ICD-10 (diagnosis)
    CPT (procedures)

These codes are essential for:
    Insurance claims
    Medical billing
    Legal and statistical reporting

Problem Statement:
The current medical coding process relies heavily on manual interpretation of clinical notes, which is time-consuming, error-prone, and costly. Inaccurate coding can lead to insurance claim rejections, delayed reimbursements, and compliance issues. There is a need for an intelligent system that can automatically analyze clinical text and assist medical coders by suggesting accurate ICD-10 and CPT codes with proper justification.

Proposed Solution ✅
We propose an AI-powered Medical Coding Automation System that:
    Accepts clinical text as input
    Uses AI + Medical Knowledge Base
    Automatically suggests:
        ICD-10 codes
        CPT codes
    Provides:
        Explanation for each code
        Confidence score
    Allows human verification
